# Nassau-Candy-Data-Analysis

A data analysis project focused on cleaning, processing, and analyzing the Nassau Candy distribution dataset to uncover key business insights and top product profitability.

## Project Overview 

This  project processes raw distribution data from Nassau Candy to identify sales trends, clean up formatting inconsistencies, and visualize profit margins across different product lines. 

## Respiratory Structure

*'Nassau Candy Distribution.csv' - The original raw dataset.
*'clean_data.py' - Python script used to clean and preprocess the raw data.
*'Cleaned_Nassau_Candy_Data.csv' - The resulting clean dataset after preprocessing.
*'process_data.py' - Python script that performs data analysis and generates insights.
*'top_products_profit.png' - A visualization chart showing the most profitable products.

## Technologies Used 

*Python 3
*Pandas (for data manipulation)
*Matplotlib / Seaborn (for data visualization)
